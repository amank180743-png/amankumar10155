# Web Technology Laboratory Assignment

This repository contains my Web Technology laboratory assignment.
The programs are organized unit-wise and demonstrate the use of
HTML, CSS, JavaScript, DOM manipulation, and Canvas
## UNIT–1: HTML
- Basic HTML page using headings, paragraphs, and text formatting tags
- Webpage demonstrating links, images, and lists (ordered, unordered, nested)
- Table-based layout with merged cells, alignment, and caption
- HTML form with text input, radio buttons, checkboxes, select menu, and submit/reset buttons
- Structured webpage using semantic elements like article, section, nav, aside, and footer
## UNIT–2: CSS
- Implementation of Inline, Internal, and External CSS
- Styling using CSS Box Model, background images, borders, margins, and padding
- Demonstration of CSS selectors: class, id, descendant, group, and universal selectors
- Responsive webpage using media queries and flexible units
- Layout design using div and span elements
## UNIT–3: JavaScript
- JavaScript-enabled webpage performing basic arithmetic operations
- Use of conditional statements and looping (for, while, do-while)
- User-defined functions and variable scope using var, let, and const
- HTML form validation using JavaScript events like onsubmit and oninput
- Alert messages for user feedback
## UNIT–4: DOM & Canvas
- DOM manipulation using getElementById(), querySelector(), and innerHTML
- Show and hide HTML elements dynamically using JavaScript
- Event handling using addEventListener for user interaction
- Simple drawing using the HTML canvas element with shapes and colors
